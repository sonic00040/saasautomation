## Error States:
- **Token expired/invalid**
- **Password mismatch**
- **Weak password** rejection
- **Network errors**
- **Rate limiting** exceeded

## Security Features:
- **Token validation** on page load
- **Single-use tokens**
- **Password strength** enforcement
- **Session invalidation**
- **Audit logging**

## Responsive Behavior:
- Mobile: Full-width form, stacked tips sections
- Tablet: Centered form with condensed side content
- Desktop: Centered layout as shown

## Flow Integration:
- **Entry**: Email reset link (with valid token)
- **Success**: Login page with success message
- **Failure**: Forgot password page or error state
- **Expired**: New reset link request
